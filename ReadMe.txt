## Resumen / Overview

es🇪🇸: Esto es un conjunto de herramientas para analizar texto en español.
El propisito es reemplazar un librería de PLN que ha tenido un tasa de errores alrededor de 5%.

en🇺🇸: This is a set of tools for working with Spanish text.
The goal is to replace an NLP library that had an error rate of about 5%.
